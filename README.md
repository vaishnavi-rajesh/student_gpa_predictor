# 🎓 Student GPA Predictor

This project predicts student GPA based on study habits and lifestyle factors using machine learning.

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning model (Regression)
- Interactive web app using Streamlit

## 📊 Dataset
The dataset is sourced from Kaggle:
https://www.kaggle.com/datasets/asifxzaman/university-students-performance-and-study-habits2026

## 🤖 Model Performance
- R² Score: 0.77
- MAE: 0.13

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
